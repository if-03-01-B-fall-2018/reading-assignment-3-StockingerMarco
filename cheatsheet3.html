Html for Lists

Start tags for

  • An ordered list

    • An ordered list starts with the < ol > tag. Each list item starts with the < li > tag. The list items will be marked with numbers by default:
    < ol >
  • Coffee

  • Tea

  • Milk

  • An unordered list

    • An unordered list starts with the < ul > tag. Each list item starts with the < li > tag. The list items will be marked with bullets (small black circles) by default:

    • Coffee

    • Tea

    • Milk

  • A description list

    • A description list is a list of terms, with a description of each term. The < dl > tag defines the description list, the < dt > tag defines the term (name), and the < dd > tag describes each term:

    Coffee

    - black hot drink

    Milk

    - white cold drink

  • List item
    • type="1"

      • The list items will be numbered with numbers (default)
    • type="A"

      • The list items will be numbered with uppercase letters
    • type="a"

      • The list items will be numbered with lowercase letters
    • type="I"

      • The list items will be numbered with uppercase roman numbers
    • type="i"

      • The list items will be numbered with lowercase roman numbers
  • A description term

    • The < dt > tag defines each term
  • A description data

    • The < dd > tag describes the term

Css for Lists

Make bullet a square
ul.b {
list-style-type: square;
}

Make bullet an image named "my-bullet.png"
ul {
list-style-image: url('my-bullet.png');
}

Make bullet an uppercase roman number
ol.c {
list-style-type: upper-roman;
}

Set bullet position to be inside the list item's text flow
ul.b {
list-style-position: inside;
}

Set bullet position to be outside the list item's text flow
ul.a {
list-style-position: outside;
}

Html for Tables

Start tags for

  • a table

    • An HTML table is defined with the < table > tag.
  • a table row

    • Each table row is defined with the < tr > tag.
  • a table header

    • A table header is defined with the < th > tag.
  • a table header cell

    • table, th, td {}
  • a table cell

    • A table data/cell is defined with the < td > tag.
  • a table body

    • < tbody >
  • a table footer

    • < tfoot >

Css for Tables

Black dotted border of 1 px width
table, th, td {
border: 1px solid black;
}

Solid border of 1 px width having a red color and rounded corners
table, th, td {
border-radius: 25px;
border: 1px solid red;
}

Merge the borders of table cells so that they appear as one line
table {
border-collapse: collapse;
}

table, th, td {
border: 1px solid black;
}
